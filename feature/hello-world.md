# Purpose 
This hello world skill serves as basic skill and allow users to start using the skills ecosystem. 

The primay purpose of this skill is to 

- User should be able to install it as first skill and test it out.
- Make user aware about skill github cli command and how to use it to install the skills 
- Use installed skills in VS Code or Github Copilot CLIs 


# Technical Details 

This skill need to be implemented by following the below steps:

- This skill will use python as default language to print hellow world in terminal if python is not available default will use OS command to print hello world in terminal.

# Technical Output 

When this feature will be implemented then a 
- VS code skills should ba avaiable in the skills directory with name `hello-world` 
- Skill should follow the skill best practices and should have requried files like `SKILL.md` and `README.md` with proper documentation.
- Skill should have a python script file which will print hello world in terminal when executed.
- Skill should have a OS spcific script filies as fall back 
- Skill shold have each OS specific script file as fall back
